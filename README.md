# SkateGenius 🛹🤖

SkateGenius is an AI-powered sports analytics platform that leverages computer vision and deep learning to analyze skateboarding performance through video-based motion tracking and pose estimation.

## Features

- Skateboard trick detection and classification
- Pose estimation and motion tracking
- Performance scoring and movement analytics
- AI-generated corrective feedback
- Progress tracking dashboard
- Video upload and analysis system

## Tech Stack

### Frontend
- React.js 

### Backend
- FastAPI / Flask
- Python

### AI & Computer Vision
- OpenCV
- MediaPipe
- YOLO
- PyTorch / TensorFlow
- NumPy & Pandas

### Database
- SQLite

## System Workflow

1. User uploads skateboarding video
2. Video frames are extracted
3. Pose estimation is applied
4. Trick classification model detects tricks
5. AI scoring engine evaluates performance
6. Feedback and analytics are displayed
